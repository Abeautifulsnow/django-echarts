Welcome to django_echarts's documentation!
==========================================

django-echarts是一个将pyecharts整合到Django开发的应用包(Django App)。主要有：

- 基于前端或后端的数据渲染
- js静态文件管理
- 数据构建工具函数库
- 基于Django命令的CLI工具
